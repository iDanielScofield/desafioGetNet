# reqres.in

## automation-api

### Tecnica aplicada
* Partição de Equivalência

### Requisitos
* Java 8+
* Gradle 6.+

### Comandos
* `gradle wrapper` - necessário para poder rodar o gradlew
* `./gradlew tasks` - mostra todas as tarefas disponíveis
* `./gradlew test` - roda todos os testes do pacote 'src/test'
